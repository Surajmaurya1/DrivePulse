# DrivePulse

A modern automotive hero section built with React, Vite, and Tailwind CSS.

## Overview

This project features a full-screen, responsive hero section designed for a car dealership or marketplace website. It utilizes a background video and decorative typography, maintaining a premium aesthetic across all screen sizes.

## Features

- Full-screen responsive hero section (minimum 600px height)
- Looping, muted background video with object-cover styling
- Gradient overlays for optimal text readability
- Decorative "NEW ERA" gradient text positioned behind the vehicle using CSS masking and z-index layering
- Responsive top navigation bar with brand logo and cart button
- Bottom Call to Action (CTA) area that adapts between desktop and mobile layouts

## Tech Stack

- React
- Vite
- Tailwind CSS v4
- Lucide React (Icons)

## Setup and Installation

1. Make sure you have Node.js installed.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Build for production:
   ```bash
   npm run build
   ```
